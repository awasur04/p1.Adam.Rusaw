Pros:

The design is very simple and easy to navigate

Star icon to represent lobby floor

Braille to assist visually impaired users

Interactive light to show which floor the user selects

Good spacing to help user understand button meanings

Labels are consistent on the left of the buttons

Physical buttons are much more reliable in case of emergency or power outage

Cons:

Bottom right has bad spacing, could be spread out more

Entire row of unused space, remove it?

Although the star is helpful, if you are unsure of the meaning there is not legend to assist

Main Elevator Interface

Each floor is a seperate button which will take you to that floor, so just press the floor you want to go to

Selected floor is highlighted in teal

Each floor has a description with what is on that floor to the left of it

The top bar is full of different buttons which all serve a different purposes:

Close Door | Open Door | Current Floor Information | Settings | Emergency Services

At the bottom of the interface is a button with braille writing which says "Tap floor number on screen"

Emergency Services

Call button to contact emergency services

Cancel button to cancel current call if it was an accident or no longer needed

Instructions are clear and concise for the user to follow in emergency

Press the fire button again to return to the main menu

Settings

Bell to ring the bell in the elevator to signal lobby staff

Power button which requires an employee code and will turn the elevator off

Light button which also requires an employee code and is used to manage the lighting inside the elevator

Toggled settings will show green or red to display the current status

Press the settings button again to return to the main menu

Staff Login

Required to toggle the power and lighting option of the settings menu

Allows input of an 8 digit code and will check to see if the code matches correct employee code (1111 for demonstration)

Backspace allows you to backup one space in case you enter the wrong code

If a correct code is inputted and the green enter button is pressed, it will return to the settings menu and apply the selected setting

Press the settings button again to return to the main menu

Tap (Accessibility Mode)

When the button is pressed below the interface to toggle accessibility mode the entire interface becomes a large button

Allows visual impaired users to tap the screen for the number of the floor they want to visit

For example, 1 tap will take them to the first floor, 2 taps will take them to the second floor, etc

You can press the button again below the screen to return to normal user mode
